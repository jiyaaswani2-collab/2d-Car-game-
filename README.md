# 2D Turbo Racer 🏎️

A classic 2D top-down car racing game built using **C++** and the **OpenGL/GLUT** library. Navigate through traffic, avoid collisions, and aim for the high score!

## 🚀 Features
* **Smooth Controls:** Responsive left/right movement.
* **Dynamic Gameplay:** Increasing difficulty as the score rises.
* **Graphics:** Rendered using OpenGL primitives and GLUT shapes.
* **Score System:** Real-time score display on the game screen.

## 🛠️ Prerequisites
To run this project, you need to have the following installed:
* **C++ Compiler** (MinGW, GCC, or MSVC)
* **OpenGL** libraries
* **GLUT / FreeGLUT** ## 🔧 Installation & Setup
